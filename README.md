# NLP LINE Bot
NTTコミュニケーションズが提供する[COTOHA API](https://api.ce-cotoha.com/contents/index.html)を利用した、手軽に自然言語処理を体験できるラインボットです。

## Description
このボットを友達追加するだけで、COTOHA APIのDevelopers Planで提供される構文解析からユーザー属性推定まで11種類を手軽に体験できます。
